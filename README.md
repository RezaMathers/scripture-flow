# 🌊 Scripture Flow

A high-retention scripture memory system designed for long-term mastery.

## 🧠 How the "Waterfall" Works
The system automatically moves verses through progressive review frequencies based on your progress:

* **Daily**: Reviewed every session for initial memorization.
* **Odd/Even**: Reviewed every other day to build muscle memory.
* **Weekly**: Assigned to a specific day of the week (e.g., all "Monday" verses).
* **Monthly**: Assigned to a specific date (1–31) for 12 deep-dive reviews a year.

## 📖 The 10 Foundational Seeds
The app initializes with these core scriptures to build a strong spiritual habit:
* **Provision & Power**: [2 Peter 1:2-3], [2 Corinthians 9:8]
* **Faith & Belief**: [Mark 9:23], [Mark 11:23-24]
* **Peace & Strength**: [Philippians 4:13], [Philippians 4:8], [Philippians 4:6-7]
* **Lifestyle & Truth**: [Ephesians 4:29], [3 John 1:2], [John 6:35]

## 🛠️ Tech Architecture
* **Sync Engine**: Real-time cloud syncing via Supabase `user_vault`.
* **Privacy**: Anonymous 6-digit access codes (no personal email required).
* **Frontend**: Built with React and Vite for blazing fast performance.
